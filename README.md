# SYNTHETICA

A website for the cyberpunk short film *Synthetica*, a Cardiff Film Production Society production.

## About the Film

Synthetica is a neo-noir cyberpunk short film about memory, grief, and identity. Written and directed by Sam Fitch.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Runtime:** Bun
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## Development

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build
```

## Fonts

- **Adversal** - Used for the main title
- **Helvetica Neue** (Ultralight/Thin) - Used for body text
